# PYTHON-DICTIONARY-SETS 🐍

Welcome to my fourth Python learning repository! This project serves as a structured collection of intermediate scripts focusing on key-value data storage and unique element collections.

The goal of this repository is to master dictionary mappings, set operations, and efficient data handling techniques as I continue my Python programming journey.

---

## 📁 Repository Structure

* **01_Dict_Sets_Basics.py** – Foundational introduction to creating, accessing, and modifying dictionaries and sets.
* **02_Dictionary.py** – Comprehensive guide to dictionary structures, accessing data, and mastering essential methods (`.keys()`, `.values()`, `.items()`, `.get()`, `.update()`).
* **03_Sets.py** – Foundation of set theory, handling unique collections, and mathematical operations.
